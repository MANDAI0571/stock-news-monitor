# 高重さん専用 統合トレンドフォロー戦略
## DUKE ＋ ウィリアム・オニール ＋ マーク・ミネルヴィニ ＋ マーケットの魔術師

作成: 2026-06-22 / 対象: 日本株 約4000銘柄 / 運用額: 300万円 / 保有: 数日〜数週間
方針: **利益最大化よりも「再現性の高さ」を優先**。勝率100%は求めない。負けトレードも学習材料。

---

## 0. この戦略のひとことまとめ

> **「強い地合いで、上昇トレンドの強い銘柄が、出来高をともなって高値圏でブレイク・または浅く押した瞬間だけを買う。損切りは必ず-7%。利は伸ばす。」**

安値株探しは一切しない。下がっている銘柄を「安いから」という理由で買うことは、この戦略では禁止。
4人の手法に共通する一点は **「強いものを買い、弱いものを避ける」**。これだけは絶対に崩さない。

---

## 1. 4つの哲学を1本に統合する

4人は言葉が違うだけで、ほぼ同じことを言っている。重なる部分が「再現性のある核」になる。

### ウィリアム・オニール（CAN-SLIM）
- **C/A**＝足元と年単位の増収増益（好決算）。**N**＝新高値・新製品・新テーマ。
- **S**＝出来高（需給）。**L**＝主導株を買う（強い銘柄）。**I**＝機関投資家の買い。**M**＝地合い（市場の方向）。
- カップウィズハンドルなどの**ベース（保ち合い）からのブレイク**を買う。

### マーク・ミネルヴィニ（SEPA / トレンドテンプレート）
- **ステージ2（上昇期）の銘柄だけ買う**。ステージ1（底ばい）・3（天井）・4（下落）は触らない。
- トレンドテンプレート: 株価>MA50>MA150>MA200、MA200が上向き、52週安値から+30%以上、52週高値の-25%以内。
- **VCP（ボラティリティ収縮）**＝出来高が枯れながら値幅が狭まった後のブレイクが理想。
- リスク管理が最優先。損切りは小さく早く。

### DUKE（新高値ブレイク投資）
- **新高値を、出来高をともなって更新した瞬間**を買う。業績の裏付けとテーマを重視。
- 決算・materialの前後で「強いものがさらに強くなる」局面を取る。日本株向けの実践型。

### マーケットの魔術師（共通教訓）
- **損切りを徹底**できる者だけが生き残る。1トレードのリスクを小さく固定する。
- **トレンドに逆らわない**。**地合いが悪いときは何もしない（現金が最強のポジション）**。
- 期待値とポジションサイジングがすべて。一発勝負はしない。

### → 統合の核（4人の最大公約数）
1. **地合い（M）が良いときだけ攻める**。悪いときは買わない。
2. **上昇トレンド（ステージ2）の強い銘柄だけ**。弱い・下落中は買わない。
3. **出来高をともなった新高値・浅い押し目**で入る。
4. **損切り-7%固定**。利は伸ばす（トレーリング）。
5. **再現性を最優先**。ルールを機械的に守り、結果を記録して学習する。

## 2. エントリー判定（毎日 約4000銘柄 → 10〜20銘柄に絞る）

3つのカテゴリに分類する。どれにも当てはまらない銘柄は **すべて見送り**。

### 【Sランク】新高値ブレイク本命（最優先・寄り付き買い候補）
高重さん指定の11条件をすべて満たす個別株（プライム/スタンダード/グロース）:

| # | 条件 | 4人の対応 |
|---|---|---|
| ① | 52週高値を更新（直近で新高値） | オニールN / DUKE |
| ② | 52週高値まで3%以内 | ミネルヴィニ(高値圏) |
| ③ | 25日線が**上向き** | ステージ2 |
| ④ | 75日線が**上向き** | ステージ2 |
| ⑤ | 株価 > 25日線 | トレンドテンプレート |
| ⑥ | 株価 > 75日線 | トレンドテンプレート |
| ⑦ | 出来高が20日平均より増加 | オニールS(需給) |
| ⑧ | 売買代金 20日平均 1億円以上 | 流動性(魔術師) |
| ⑨ | 好決算（増収増益 or 進捗良好） | オニールC/A |
| ⑩ | 上方修正あり | サプライズ買い |
| ⑪ | 強いテーマ（AI/半導体/防衛/DC/電力/ロボット/インフラ/国策） | オニールN/L |

補強（あればSの確度が上がる。必須ではない）: 200日線も上向き＆株価>200日線（完全なステージ2）、
VCP/カップウィズハンドルのブレイク、機関の買いを示す大商い日。

→ **Sは「翌営業日の寄り付きで買う候補」**。だからこそ決算リスクが残る銘柄はSにしない（後述5）。

### 【押し目候補（Aの中核）】上昇トレンド継続中の浅い押し
強いトレンドが続く銘柄が、過熱を冷ましてから再上昇する局面を取る:
- 25日線にタッチ、または25日線から+5%以内
- 直近高値から3〜20%の調整（深すぎる押しは取らない＝トレンド崩れの疑い）
- 25日線・75日線がまだ上向き＝トレンド継続中
- 出来高は押し目で枯れ、反発で増えるのが理想（VCPの考え方）

### 【長期押し目候補（監視→反発確認でA/B）】
- 200日線に接近、または200日線で反発の初動
- 200日線が上向きで、株価が200日線を割り込んでいない（割れたら見送り）
- 業績悪化なし（赤字転落・下方修正が出ていない）
- → これは「待ち伏せ」。反発を確認してからエントリー（落ちるナイフは掴まない）

### 除外条件（1つでも該当したら無条件で外す）
赤字企業 / 債務超過 / 出来高不足 / 低流動性（売買代金1億未満） / 仕手株（材料不明の急騰・値幅異常）/
ETF / ETN / REIT / 投資法人 / インフラファンド。
さらに: 決算14営業日前〜決算翌営業日（決算ギャンブル回避＝後述5）。

### 毎日10〜20銘柄に絞るルール
1. 除外条件で母集団をふるい落とす
2. S/押し目/長期押し目に分類
3. スコア順に並べ、**S → 押し目上位 → 長期押し目上位** の順で上から最大20銘柄まで
4. Sが20を超える強い地合いの日は、スコア＋テーマ強度＋出来高増勢で上位20に絞る
5. Sがゼロの日は「該当なし＝買わない」を正直に出す（無理に埋めない）

## 3. 地合いフィルター（最優先・全条件に優先する）

魔術師とオニールの「M」。**地合いが悪い日は、どんなに良い銘柄が出ても買わない。**

| 地合い | 状態 | 行動 |
|---|---|---|
| NORMAL | 平常 | 通常運用（最大3枠まで建てる） |
| CAUTION | 警戒 | 新規は厳選（Sのみ・枠を絞る） |
| RISK | 危険 | 原則新規見送り、保有は損切りライン厳格化 |
| STOP | 全停止 | 新規買い停止・現金保有（台湾有事/暴落など） |

- 自動判定: 日経・TOPIX・米国指数（NASDAQ/S&P）・VIX の前日比で点数化。
- 手動オーバーライド: **GitHub の `regime.txt` の1語をスマホで書き換えるだけ**で即 RISK/STOP に上げられる。
  （市場が織り込む前に、高重さんの現場感覚で守りに入れる安全装置）
- 米国市場急落時は「何を買うか」より「何を買わないか」。全額投入は絶対にしない。

## 4. 売買ルール（機械的に守る＝再現性の生命線）

### 買い
- 買うのは **Sランク**、地合いに余裕があれば **押し目候補の上位** まで。
- 寄り付き成行 or 前日終値近辺の指値。追いかけ高値掴みはしない（高値から離れたら見送り）。
- **1トレードのリスクを資金の約2%に固定**（後述サイジング）。

### 損切り（最優先・例外なし）
- **エントリーから-7%で機械的に損切り**。これは絶対。「戻るはず」で延ばさない。
- ブレイク銘柄がブレイク地点を割り込んだら、-7%を待たずに撤退してもよい（ミネルヴィニ流の早逃げ）。

### 利確（利は伸ばす）
- **+20%に到達したら**、そこから**トレーリングストップ**に切り替える:
  - 例: 高値から-8%、または25日線割れ で手仕舞い。
- +20%未満の段階では、25日線を明確に割るまで保有を継続（トレンドが続く限り持つ）。
- 保有上限 **10〜20営業日のタイムアウト**目安（数日〜数週間の方針に合わせる）。動かない資金は次の候補へ。

### 決算をまたがない
- 保有中に決算が**2営業日以内**に近づいたら、原則として引けで一旦手仕舞い（決算ギャンブル回避）。

### サイジング（300万円・最大3枠）
- 規律版: 最大3銘柄・1枠100万円・Sランクのみ・S不足は現金。
- リスクベースの目安: 1トレードの損失上限 ＝ 300万×2% ＝ **6万円**。
  -7%損切りなら 1枠 ≒ 6万 ÷ 0.07 ≒ **約86万円**＝1枠100万は妥当な範囲。
- 地合いが良く成績が良いほど枠・銘柄数を増やしてよいが、**初期は守りから**。成績で攻め度を上げる。

## 5. 「Sランク＝寄り付き買い候補」だから決算リスクを排除する

Sは翌朝の寄りで買う。だから**決算サプライズの事故**を最も嫌う。
- 決算予定日が確認できない銘柄（決算未確認）は **最大Aまで**。Sにはしない。
- 決算14営業日前〜決算翌営業日は **除外**。
- 「安川電機ショック」のような決算跨ぎの急落を、寄り買い候補からは構造的に締め出す。

## 6. 検証で確かめること（バックテスト／詳細は別ファイルのエンジン）

過去5年で、勝率・平均利益・平均損失・PF・最大DD・期待値・保有期間別（5/10/20/40日）を計測。
さらに10項目を分析: 勝ちパターン共通点 / 負けパターン共通点 / 最も利益が出る条件 /
地合いとの関係 / 日経・TOPIX・グロースとの相関 / セクター別 / 出来高別 / 時価総額別。

検証で見たいのは「儲かったか」ではなく、**「どの条件のとき再現性高く勝てるか」**。
勝率が多少低くても、期待値がプラスで・最大DDが許容範囲で・条件が明確なら採用する。
逆に高勝率でも、たまたま・条件が曖昧・大負けが混じるものは捨てる。

> ⚠️ 数字はすべて高重さんのMacで実データを回してから埋める。
> クラウド側（私）は株価に接続できないため、数字は捏造しない。空欄は「未検証」と正直に出す。

## 7. 運用ループ（毎日・スマホ完結を目標）

1. （引け後・Mac）約4000銘柄をスキャン → S/押し目/長期押し目に分類 → 上位10〜20銘柄
2. （翌朝・Mac→Gmail）地合い＋10〜20銘柄を朝メール。各銘柄に
   **買い理由 / 損切り位置（-7%）/ 目標株価 / 期待値 / 過去類似パターン** を表示
3. （スマホ）高重さんが地合いと出来高を見て最終判断（買う/見送る）
4. 建てたら trade_journal に記録。手仕舞いも記録 → pattern_learn が勝ち負けを学習
5. 週末: 週間レポートで「成功・失敗・対策・調査」を振り返る

## 8. やらないことリスト（再現性を壊す行動）

- 地合いが悪い日に「良い銘柄があるから」と買う
- 下落中・ステージ4の銘柄を「安い」「割安」で買う
- 損切り-7%を「戻るはず」で延ばす
- 材料不明の急騰株・仕手株に飛び乗る（イナゴタワー）
- 決算を勝負のつもりでまたぐ
- 1銘柄に資金を集中させて一発を狙う
- ルールを気分で変える（ルール変更は検証してから・記録を残す）
