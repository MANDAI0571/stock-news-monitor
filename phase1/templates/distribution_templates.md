# 配信文テンプレート（4種）

すべてプレースホルダ `{{...}}` を実データで置換してから使う。置換前のテンプレは外部送信しない。
公開前に `phase1/tools/compliance_check.py` を必ず通す（断定・推奨・利益保証・禁止トークン・プレースホルダ・免責を機械チェック）。
数値は当日のGmailスクリーニング由来の実データのみ。取得できない項目は行ごと省略（捏造しない）。

共通プレースホルダ:
`{{DATE_JP}}`=2026年7月17日 / `{{REGIME}}`=地合い / `{{TOP_CODE}}`=筆頭銘柄コード / `{{TOP_NAME}}` / `{{TOP_PRICE}}` / `{{REACHED_N}}`=到達・接近件数 / `{{NOTE_URL}}` / `{{SIGNUP_URL}}`=登録導線(UTM付き) / `{{CHART_URL}}`=https://finance.yahoo.co.jp/quote/{{TOP_CODE}}.T/chart

UTM: `{{SIGNUP_URL}}` は必ず `?utm_source=..&utm_medium=..&utm_campaign=phase1_YYYYMM&utm_content=..` を付与（規約は phase1_collaboration.md）。

## ① X（SNS）テンプレート — 短く、結論と銘柄を先に

トーン: 結論ファースト。1投稿140字前後。必要ならスレッドで補足。断定・推奨語を使わない。

```
【{{DATE_JP}}｜52週新高値スキャン】
到達・接近 {{REACHED_N}}銘柄。地合いは{{REGIME}}。
注目は {{TOP_NAME}}（{{TOP_CODE}}）＝現在値{{TOP_PRICE}}円・52週高値に接近。
まとめはこちら↓
{{SIGNUP_URL}}
#日本株 #52週新高値 #新高値ブレイク
```

スレッド2通目（任意・補足）:
```
本命の並び（52週高値差／出来高倍率）:
・{{S1_NAME}}（{{S1_CODE}}）{{S1_DIST}}／{{S1_VOL}}倍
・{{S2_NAME}}（{{S2_CODE}}）{{S2_DIST}}／{{S2_VOL}}倍
・{{S3_NAME}}（{{S3_CODE}}）{{S3_DIST}}／{{S3_VOL}}倍
※観察記録です。売買推奨ではありません。
```

## ④ note CTA フッター テンプレート — 記事末尾（免責の直前）に固定挿入

note本文は「運用記録として読み物形式」（既存の生成フォーマット）。配信に必要なのは登録導線。
毎記事、ハッシュタグの後・免責の前に次のブロックを固定で入れる。

```
---

📩 毎朝の52週高値スキャンを受け取りたい方へ
その日の到達・接近銘柄を、実データのまとめでお届けしています。
登録はこちら → {{SIGNUP_URL}}

---
```

## 使い方メモ

- 数値・銘柄はその日のスクリーニング実データで置換。取得できない項目（例: 売買代金）は該当行を丸ごと省く。
- `{{TOP_TURNOVER_LINE}}` のように「あれば足す」項目は、値が無ければ空文字にする。
- 置換後、各配信文（X／LINE／メール）とnoteを `python3 phase1/tools/compliance_check.py --input <file>` に通し、exit 0 を確認してから配信下書きに回す。
- 同じ記事をClaude系とCodex系の両方から配信しない（重複防止は pre_publish_checklist.md でチェック）。
